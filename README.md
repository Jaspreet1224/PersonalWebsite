# PersonalWebsite

A personal portfolio website with:
- Projects section
- Resume page
- About section
- Profile picture area
- Contact links

## Files
- `index.html`: Main portfolio page
- `resume.html`: Detailed resume page
- `styles.css`: Site styling
- `script.js`: Mobile menu + reveal animations
- `assets/profile-placeholder.svg`: Default image placeholder

## Run locally
Open `index.html` in your browser.

## Customize quickly
1. Replace `Your Name` and `you@example.com` in `index.html` and `resume.html`.
2. Replace `assets/profile-placeholder.svg` with your own photo (for example `assets/profile.jpg`) and update the image path in `index.html`.
3. Add your real project details and links in the Projects section.
4. Update resume entries in `resume.html` and the resume summary in `index.html`.
5. Update LinkedIn and GitHub URLs in the Contact section.
